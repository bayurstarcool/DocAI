# Dataset sample yang direkomendasikan

1. DocShadow-SD7K — dataset utama document shadow removal. Berisi pasangan dokumen high-res dengan shadow dan shadow-free.
2. ISTD / ISTD+ — triplet image shadow, mask, shadow-free. Tidak khusus dokumen, tapi bagus untuk pretraining mask.
3. SRD / SRD+ — shadow removal umum, cocok untuk pretraining.
4. Doc3D — synthetic document dewarping, untuk tahap meluruskan dokumen melengkung.
5. DocUNet Benchmark — evaluasi dewarp real document.
6. RVL-CDIP / IIT-CDIP — clean document source untuk synthetic shadow generator.

Catatan: cek lisensi masing-masing dataset sebelum dipakai komersial.
